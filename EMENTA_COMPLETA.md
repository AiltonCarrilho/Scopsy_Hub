# EMENTA COMPLETA - Scopsy Lab

Estrutura Curricular para População de Dados
Versão: 1.0

**Objetivo:** Guiar a criação progressiva e ordenada de casos clínicos
**Baseado em:** Progressão pedagógica (Bloom's Taxonomy + ZDP)

---

## 🎯 PRINCÍPIOS DE ORDENAÇÃO

**Critérios de Progressão:**

- Fundamentos → Aplicação
- Simples → Complexo
- Reconhecimento → Produção
- Protocolo → Adaptação
- Individual → Sistêmico

**Níveis de Dificuldade:**

- **NÍVEL 1 (Iniciante):** Informação explícita, uma variável
- **NÍVEL 2 (Básico):** Informação clara, 2-3 variáveis
- **NÍVEL 3 (Intermediário):** Ambiguidade moderada, 3-5 variáveis
- **NÍVEL 4 (Avançado):** Alta ambiguidade, comorbidades
- **NÍVEL 5 (Expert):** Casos atípicos, dilemas éticos

---

### MÓDULO 1: CONCEITUAÇÃO DE CASO (8-10h)
