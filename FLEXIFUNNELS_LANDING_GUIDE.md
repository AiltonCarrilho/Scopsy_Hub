## 🚀 PUBLICAÇÃO

### **PASSO 1: Preview**
1. FlexiFunnels → Preview
2. Testar em Desktop e Mobile
3. Clicar em TODOS os botões
4. Verificar links

### **PASSO 2: Configurar Domínio**
1. Settings → Custom Domain
2. Adicionar: scopsy.com.br
3. FlexiFunnels vai pedir DNS (CNAME ou A)
4. Copiar instruções

### **PASSO 3: DNS no Cloudflare**

**Se FlexiFunnels pedir CNAME:**
```
Type: CNAME
Name: @
Target: [fornecido pelo FlexiFunnels]
Proxy: OFF (cinza)
```

**Se FlexiFunnels pedir A Record:**
```
Type: A
Name: @
Target: 137.66.21.168 (já está configurado!)
Proxy: OFF (cinza)
```

### **PASSO 4: Publicar**
1. Verificar tudo
2. Clicar em "Publish"
3. Aguardar 5-10 min (DNS)
4. Testar em: https://scopsy.com.br
